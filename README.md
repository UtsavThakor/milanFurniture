# Milan Furniture

Modern responsive website for Milan Furniture, Bharuch — 
Gujarat's trusted furniture destination since 1998.

Built with: HTML · CSS · Vanilla JavaScript

Features:
- Mobile-first responsive design (3 breakpoints)
- Image settings panel with localStorage persistence
- CSS grid category layout
- Marquee animation
- Featured products section
- Netlify deployment

Live: https://milan-furniture.netlify.app
